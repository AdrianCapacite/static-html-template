Example HTML template for a static website.

## Example structure of a static website

```plaintext
.   # Root directory
├── index.html
├── about.html
└── assets/
    ├── images/
    │   ├── example.jpg
    │   └── example2.jpg
    ├── js/
    │   └── main.js
    └── style/
        └── main.css
```
